Dès qu'une nouvelle page est publiée ou créée sur automationact.com, ajouter dans TASKS.md sous "n8n - Workflows à construire" la tâche : `[ ] Construire le workflow n8n pour **[nom de la page]** (post WP #XXXX)`.

**Why:** L'utilisateur veut que chaque page de contenu ait son workflow n8n associé — la page est la promesse, le workflow est la livraison.

**How to apply:** Dès qu'un post WP est mentionné ou créé (ID, titre, URL), créer immédiatement l'entrée n8n dans TASKS.md sans attendre qu'il le demande.
