Toujours afficher l'évaluation fit/no-fit avec des bullets ✅/⚠️/❌, un point par ligne, sans tableau ni titres en gras. Format validé par l'utilisateur :

```
Points forts / Points faibles :

✅ SEA Google Ads + Bing Ads → core skills
✅ Gestion multi-clients (freelance, DiliTrust multi-continents)
✅ Agents IA paid media → différenciateur sur "campagnes automatisées"
✅ 6+ ans paid → correspond au "5 ans ou plus"
⚠️ SA 360 → non utilisé, outil central de l'offre
⚠️ Expérience en agence → D-egde uniquement (1 an, 2019), le reste côté annonceur
⚠️ Management d'équipe récent → absent du CV
```

**Why:** Permet une lecture rapide. Pas de prose, pas de tableau, pas de titres séparés "Points forts" / "Points faibles".

**How to apply:** Utiliser ce format dès l'étape 2b (évaluation fit) pour chaque candidature, qu'elle soit NO-GO ou candidatable. ❌ pour les blockers structurels, ⚠️ pour les lacunes assumables, ✅ pour les points forts.
